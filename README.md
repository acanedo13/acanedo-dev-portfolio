# AC_PORTFOLIO // Adriana Canedo

### OVERVIEW
A high-performance, responsive portfolio site designed for a Software Engineer & UI/UX Design Analyst. Built as a single-page application (SPA) with a terminal-inspired aesthetic to reflect a background in defensive cyber operations and systems architecture.

### TECH STACK
- **Framework:** React.js
- **Design System:** Chakra UI
- **Animations:** Framer Motion (Scroll progress, fade-in transitions)
- **Icons:** React Icons (Fa, Si)
- **Deployment:** Vercel

### KEY FEATURES
- **Interactive Project Demos:** Custom Modal-based video walkthroughs for deep-dive project presentations.
- **Dynamic Stack Filtering:** Real-time search/filtering for technical skills.
- **Responsive Architecture:** Built with a mobile-first approach using Chakra UI grid systems.
- **Immersive UX:** Integrated PDF viewer for seamless resume access and motion-heavy interface elements.
